## About This Application

This is a roadmap planning and management tool for Notable's AI workflow automation platform. The application enables healthcare organizations to plan, prioritize, and track automation flows across their operations.

**Key Features:**

- **Roadmap Planning**: Visual roadmap for managing flows through Draft, Review, and Deployed stages
- **Value Analysis**: Calculate and visualize ROI, total value, and opportunity ratings for each automation flow
- **Scatter Plot View**: Prioritize flows based on complexity vs. value to identify high-impact opportunities

The app helps stakeholders evaluate automation opportunities by tracking key metrics including baseline volume, completion rates, value per run, and implementation complexity.
